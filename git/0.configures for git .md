## How to add alias commands in git?
-  Edit ~/.gitconfig  
<pre>
[alias]
  gst = 'git stataus'
  gcm = 'git commit --m
  gca = 'git commit --ammed
</pre>
or use git config alias command:
```bash
$ git config --global alias.gst  'git status'
```
